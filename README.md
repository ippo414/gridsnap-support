# GridSnap Support / サポート

GridSnap is a grid-based window manager for macOS.
GridSnapは、macOS用のグリッドベース・ウィンドウマネージャーです。

## Support / お問い合わせ

- Bug reports and feature requests / 不具合報告・機能要望：[Issues](https://github.com/ippo414/gridsnap-support/issues)

## FAQ

**Q. Windows don't move / ウィンドウが動きません**
Grant Accessibility permission in System Settings > Privacy & Security > Accessibility.
システム設定 > プライバシーとセキュリティ > アクセシビリティ で GridSnap を許可してください。

**Q. How do I show the grid? / グリッドの表示方法は？**
Double-tap the Command key, or press Ctrl+Option+G.
Cmdキーをダブルタップ、または Ctrl+Option+G を押してください。
